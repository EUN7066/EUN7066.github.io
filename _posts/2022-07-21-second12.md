#  K번째 약수


```python
# 어떤 자연수 p와 q가 있을 때, 만일 p를q로 
나누었을 때 나머지가 0이면 q는 p의 약수이다.
두 개의 자연수 N과 K가 주어졌을 때, 
N의 약수들 중 k번째로 작은 수를 출력하는
프로그램을 작성하시오
```


```python
n, k=map(int, input().split())
cnt=0
for i in range(1, n+1):
    if n%i==0:
        cnt+=1
    if cnt==k:
        print(i)
        break
else:
    print(-1)
```

    3
