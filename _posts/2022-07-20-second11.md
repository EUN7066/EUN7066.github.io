# 람다 함수 


```python
def plus_one(x):
    return x+1

print(plus_one(1))
```

    2
    


```python
# 위의 코드를 람다 함수로 표현
```


```python
plus_two=lambda x: x+2
print(plus_two(1))
```

    3

```python

print(list(map(lambda x: x+1, a)))
```
